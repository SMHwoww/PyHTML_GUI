# PyHTML 未来发展规划

本文档详细规划了PyHTML项目的未来发展方向，包括短期、中期和长期目标，以及技术路线和功能扩展计划。

## 项目现状

目前PyHTML项目已经实现了以下功能：

- 基于PyQt5的图形界面
- 组件化HTML生成
- 实时预览功能
- 多种内置组件（标题、走马灯、文章、图片、分割线、按钮、引用、导航栏、卡片、列表、表单、视频）
- 组件属性编辑
- HTML导出功能
- 项目保存和加载
- 组件导入功能
- HTML Head设置
