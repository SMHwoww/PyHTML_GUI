# 主题系统规范

## 1. 概述

本规范定义了PyHTML GUI的主题系统，包括主题的结构、管理方式、预设主题以及自定义主题的实现方法。

## 2. 目标

- 实现可切换的主题系统
- 支持自定义主题
- 提供多种预设主题
- 确保主题在整个应用中一致应用
- 支持实时主题切换

## 3. 架构设计

### 3.1 主题管理模块

创建一个新的模块 `src/core/theme.py`，负责主题的加载、保存和管理。

### 3.2 主题结构

主题采用JSON格式存储，包含以下结构：

```json
{
  "name": "主题名称",
  "description": "主题描述",
  "author": "作者",
  "version": "1.0",
  "colors": {
    "primary": "#4CAF50",         // 主色调
    "secondary": "#333333",       // 次要色调
    "background": "#f0f0f0",       // 背景色
    "text": "#333333",            // 文本颜色
    "text_light": "#ffffff",       // 浅色文本（用于按钮等）
    "text_black": "#000000",       // 黑色文本（用于主界面）
    "input_background": "#ffffff",  // 输入框背景色
    "menu_text": "#ffffff",        // 菜单栏文本颜色
    "border": "#dddddd",           // 边框颜色
    "hover": "#45a049",            // 悬停颜色
    "pressed": "#3d8b40",          // 按下颜色
    "error": "#f44336",            // 错误颜色
    "success": "#4CAF50",          // 成功颜色
    "warning": "#ff9800",           // 警告颜色
    "info": "#2196f3"               // 信息颜色
  },
  "fonts": {
    "family": "Microsoft YaHei",
    "size": 12
  },
  "styles": {
    "main_window": "",
    "widget": "",
    "label": "",
    "push_button": "",
    "list_widget": "",
    "line_edit": "",
    "menu_bar": "",
    "menu": "",
    "status_bar": "",
    "custom_title_bar": "",
    "title_label": "",
    "title_button": ""
  }
}
```

### 3.3 主题文件存储

- 预设主题存储在 `src/core/themes/` 目录下
- 用户自定义主题存储在用户配置目录中

## 4. 实现细节

### 4.1 主题管理类

创建 `ThemeManager` 类，负责：

- 加载预设主题
- 加载用户自定义主题
- 切换主题
- 保存自定义主题
- 生成样式表

### 4.2 主题切换功能

在主窗口的设置菜单中添加主题选择选项，包括：

- 预设主题列表
- 自定义主题选项
- 主题编辑器

### 4.3 组件主题支持

修改组件系统，使组件能够：

- 响应主题变化
- 根据当前主题调整样式
- 支持主题变量

### 4.4 实时主题切换

实现实时主题切换功能，当用户选择新主题时，立即应用到整个应用。

## 5. 预设主题

提供以下预设主题：

1. **默认主题**：当前的绿色主题
2. **暗色主题**：深色背景，适合夜间使用
3. **蓝色主题**：蓝色为主色调
4. **紫色主题**：紫色为主色调
5. **高对比度主题**：适合视觉障碍用户

## 6. 自定义主题

### 6.1 主题编辑器

创建主题编辑器对话框，允许用户：

- 修改主题颜色
- 调整字体设置
- 预览主题效果
- 保存为新主题

### 6.2 主题导入/导出

支持主题的导入和导出，便于用户分享主题。

## 7. 集成到现有系统

### 7.1 主窗口修改

修改 `MainWindow` 类，添加：

- 主题管理实例
- 主题切换菜单
- 主题应用方法

### 7.2 项目配置

在 `Project` 类中添加主题配置，使项目能够保存当前主题设置。

### 7.3 组件系统修改

修改 `Component` 类，使其能够：

- 接收主题信息
- 根据主题调整样式

## 8. 技术实现要点

- 使用Qt的样式表系统实现主题
- 采用JSON格式存储主题配置
- 实现主题缓存，提高性能
- 支持主题预览功能
- 确保主题切换时的平滑过渡
