# pyHTML - 报纸组件化HTML生成器

一个用于快速创建报纸风格HTML页面的Python工具，支持组件化开发、图形化界面和实时预览。

## 功能特点

- **组件化开发**：将报纸内容拆分为可复用组件
- **图形化界面**：基于PyQt5的直观GUI
- **实时预览**：快速查看HTML渲染效果
- **HTML/JS生成**：生成可直接部署到CloudFlare的静态文件
- **第三方扩展**：支持加载自定义组件
